Core features
01
Markdown
Write in markdown with live split preview. Syntax highlighting, headings, lists, blockquotes.
02
Drawing
One shared canvas per pad. Every stroke syncs in real time to every connected peer.
03
Live files
Drop files into a pad. Everyone grabs them. No upload, no storage. Files live with the session.
04
Presence
See who's editing. Cursors, names, connection status. You feel the collaboration.
05
Persistence
Text and drawings auto-save. Come back later, everything is still there. No save button.
06
Pad trees
Pads live at paths. Paths create natural hierarchy. Navigate related pads by structure.
File sharing view
How it works
1
Open a URL. Any path becomes a pad. No sign-up, no install, no waiting.
2
Write and draw. Markdown on the left, preview on the right. Switch to the canvas to sketch.
3
Share the link. Anyone with the URL joins instantly. You see their cursor, they see yours.
4
Drop files. Drag a file in and everyone connected can download it. No cloud, no limits.
