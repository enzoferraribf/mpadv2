// what you get
Markdown
Write in markdown with a live split preview. Syntax highlighting, headings, lists, links, images, blockquotes — rendered as you type.
Drawing
One shared drawing surface per pad. Sketch diagrams, annotate ideas, or just doodle. Everyone sees every stroke in real time.
Live files
Drop files into a pad and they're instantly available to everyone connected. Files live as long as the session — no storage, no cleanup.
Presence
See who's in your pad. Cursors, names, connection status. You feel the others without any setup or sign-in.
Persistence
Text and drawings persist automatically. Come back tomorrow — your work is still there. No save button, no cloud accounts.
